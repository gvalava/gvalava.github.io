About This Page

This page is a no-frills demonstration of some concepts using vanilla front end web technologies.

HTML Semantic elements are used to organize this page (header, nav, main, aside, footer), demonstrate different kinds of lists and where they are suited (ol, dl), and to present content outside the usual text properly (figure, code).
CSS The CSS for this page uses a grid layout to create a responsive design that places the aside element to the right of the main and nav elements in wider screens as well as pseudo-classes (:hover) for changes in element appearance.
JavaScript The links in the nav element were generated by JavaScript, which also added the id attributes to make the links work. A button was added to the page, with inline JavaSccript that shows a dialog box with the specified information as per instruction.
